Mini Projet

Le mini projet vise à nous préparer pour le projet final.

Les étapes du Miniprojet

Au départ, le programme devrait afficher les adresses URL d'un fichier texte. Ensuite, analyser chaque URL et afficher le code HTTP, l'encodage de la page et le nombre des mots. Les resultats devraient être stockés dans un tableau HTML. Puis, la sortie a été modifiée pour enregistrer les résultats dans un fichier HTML. Enfin, le programme a été ajuster pour gérer cette page web avec stylesheets Bulma.
